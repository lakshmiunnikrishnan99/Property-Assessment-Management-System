# Property Assessment Management System

## Overview

This project demonstrates relational database design and SQL querying concepts learned through **BCIT COMP 1630 – Relational Database Management Systems and SQL**

The database models a fictional property assessment system that manages municipalities, property owners, assessments, property types, and sales transactions.
